# valoratuempresa.es

Plataforma de valoración de empresas online para PYMEs españolas.

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Despliegue en Vercel

El proyecto se despliega automáticamente al hacer push a la rama `main` en GitHub.
